MODE DESCRIPTIONS
-----------------
A  - Restore → Upscale → Upscale
B  - Restore Soft → Upscale → Upscale
C  - Denoise + Upscale → Upscale
A+ - Restore → Upscale → Restore → Upscale
B+ - Restore Soft → Upscale → Restore Soft → Upscale
C+ - Denoise → Restore → Upscale

QUALITY DESCRIPTIONS
--------------------
S (FAST)      - Low overhead, mobile devices
M (BALANCED)  - Medium quality, general use
L (HIGH)      - Best quality, high-end devices

GPU COMPATIBILITY
-----------------
- OpenGL: Uses legacy 'gpu' backend + opengl-pbo
- Vulkan (Android 13+): Can use 'gpu-next' backend
- gpu-next + OpenGL: Incompatible (disabled)
